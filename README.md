# Airclean_board_Knowledge_base

空气净化器控制板硬件资料库，不是可直接编译的固件工程。

## 入口

- `Airclean_board_Context7_KB.md`：已核对的硬件事实、协议、引脚和固件边界。
- `context7.json`：Context7 解析配置与回答规则。
- 原始 PDF、Excel、原理图和 PCB 图：需要追溯或补充验证时使用。

`Airclean.json` 含发布凭据，已排除在索引和提交范围外；不得复制、转发或写入文档。

## 推荐阅读

1. 先读 `Airclean_board_Context7_KB.md`。
2. 改板级逻辑前，核对 `空气净化器管脚图.xlsx` 与原理图。
3. 按功能查原始资料：显示查 TM1628A，语音查 SU-03T，联网查涂鸦协议，触摸查触摸手册。
4. 不确定项以实板、BOM 和波形验证为准。

## 已确认的关键结论

- `U2` 为 `CMS79F723`；四个触摸键分别为 `RA3/TK3` 定时、`RA0/TK0` 风速、`RA2/TK2` 开关、`RA1/TK1` 滤网重置。
- TM1628A 使用 `RA4=DIO`、`RA5=CLK`、`RA6=STB`。本板段线不是标准顺序，不能直接套用 `0x3F` 字模。
- `U5` 语音、`U6` WiFi 与 `H5` 共用 `RC0/RC1` 网络；未确认前不要让两个外设同时发送。
- 涂鸦协议使用 `9600, 8N1`、无流控和 `55 AA` 帧头；`101`~`105` 可下发/上报，`106`、`107` 只上报。
- `RB3` 经 Q10 低有效控制 `FAN`，不是硬件 CCP/PWM；应按软件 PWM 和实机标定实现。
- `RB2/AN13` 接 `H3:SMELL`，现有资料不能证明它是 NTC 温度通道。
